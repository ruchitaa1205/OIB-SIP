# Car Price Prediction

## Objective
Build a machine learning model to predict the price of a car based on its features.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Files
- Car_Price_Prediction.ipynb
- car_data.csv
- requirements.txt

## Project Description
This project uses regression algorithms to predict car prices based on features such as year, fuel type, transmission, and mileage.
