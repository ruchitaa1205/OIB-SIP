# Iris Flower Classification

## Objective
Build a machine learning model to classify Iris flowers into Setosa, Versicolor, and Virginica.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Files
- Iris_Flower_Classification.ipynb
- iris.csv
- requirements.txt

## Result
The model predicts the Iris flower species with high accuracy.
